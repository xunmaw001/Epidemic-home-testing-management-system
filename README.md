# Epidemic-home-testing-management-system
疫情居家检测管理系统是一款基于Spring Boot和Vue.js开发的综合性管理平台，旨在提升疫情防控效率。系统通过用户注册、居家健康打卡、检测信息上报与审核等功能，实时监控居家人员健康状况，并通过数据分析提供决策支持。其友好的用户界面和高效的数据处理能力，帮助政府和社区有效管理疫情，确保公众安全与健康。
